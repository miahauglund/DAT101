    4. [Hvis en funksjon ikke returnerer noe, vil en variabel som prøver å lagre funksjonens resultat få verdien undefined. Dette skyldes at funksjoner automatisk returnerer undefined når ingen returverdi er spesifisert.]

  4. [How do we separate parameters from each other when creating a function?]

    5. [Når man definerer en funksjon, skilles parametere med komma inne i parentesene. Eksempelvis i function eksempel(param1, param2), er param1 og param2 adskilte parametere]

  5. [Explain how parameter names are related during the invocation and creation of a function]

    6. [Ved opprettelse fungerer parameternavn som plassholdere for verdier som sendes inn senere. Når funksjonen kalles, sendes verdier (argumenter) i samme rekkefølge, slik at de samsvarer med parametrene.]

  6. [Explain why a parameter is undefined when a function call is missing a parameter.]

    7. [Hvis man kaller en funksjon uten å sende inn alle argumenter, blir manglende parametere automatisk undefined. Dette er fordi det ikke gis noen verdi til disse parameterne]
